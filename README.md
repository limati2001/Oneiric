# Oneiric v0.0.1 initial push (Minecraft Alpha 1.1.2_01 Mod)


## Features of the Nothing Burger Release
- Pratically Nothing
- Pastel Pink/Lavender Skies (`0xD4A0C8`)
- Warm Pastel Fog (`0xE8B4A0`)
- Soft Cloud Colors (`0xF5D5E8`)

## How to Install
1. Download the `Oneiric-v0.0.1.zip` from the **Releases** tab.
2. In Prism Launcher, open your Alpha 1.1.2_01 (make sure you have it downloaded) instance settings.
3. Go to **Version** -> **Add to minecraft.jar** and select the downloaded zip.
4. Move the jar mod up to top
5. Play
