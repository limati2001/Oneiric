# Oneiric v0.0.2 (A Minecraft Alpha 1.1.2_01 Mod)


## Features of the v0.0.2 snapshot
- Bug fix: alpha leaf block decayin trees
- Window and GUI title changed/removed
- Major changes in world gen. to more vast landscapes and less liquid gen.
- Cactus removed. Entirely.


## How to Install
1. Download the `Oneiric-v0.0.2.zip` from the **Releases** tab
2. In Prism Launcher, open your Alpha 1.1.2_01 (make sure you have it downloaded) instance settings
3. Go to **Version** -> **Add to minecraft.jar** and select the downloaded zip
4. Move the jar mod up to top
5. Play
